Platform Modules

Five modules, one pipeline

Each module handles a distinct stage of the genomic selection workflow.

01

Chip Data Imputation

Variant-Specific SNV

Reconstructs missing genotype calls from sparse SNP chip panels using variant-specific imputation models, recovering higher-density genomic resolution without a full re-genotyping cost.

02

Quality Control

MAF · HWE · Call Rate

Filters genotype data on Minor Allele Frequency thresholds, tests markers for departure from Hardy-Weinberg Equilibrium, and enforces per-sample and per-marker call rate cutoffs before any downstream analysis runs.

03

Genomic Prediction Module

GEBV Estimation

Estimates genomic breeding values directly from marker data, giving breeders a selection signal earlier and with greater accuracy than pedigree-based methods alone.

04

Model Selection

GBLUP · BayesB · BayesR

Lets users choose and compare prediction models — GBLUP, Bayes B, or Bayes R — to match the genetic architecture of the trait being selected for, from polygenic to major-gene-driven.

05

Mate Allocation Service

Optimal Mating

Generates recommended mating pairs that balance expected genetic gain against inbreeding risk, turning prediction output into an actionable breeding plan.
